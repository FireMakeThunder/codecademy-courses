Tag

Description

Attributes
<img> Displays an image.
  • src: URL reference to the resource.
  • alt: Defines an alternative text description of the image.
  • width: Rendering width of the image in pixels. Must be an integer without a unit.
  • height: Rendering height of the image in pixels. Must be an integer without a unit.
<Picture> HTML5 element used to provide multiple image sources for rendering depending on the display/device scenario. See <source> as a related tag.
  • This element includes only global attributes.
<video> HTML5 element used to render a video onto the document See <source> as a related tag.
  • src: URL reference to the resource. You may also use <source> instead.
  • controls: A boolean value to display controls for this element.
  • autoplay: A boolean element to autoplay when rendered.
  • loop: A boolean element to loop playback.
<audio> HTML5 element used to include an audio file into the document. This is not a self-closing tag. See <source> as a related tag.
  • src: URL reference to the resource. You may also use <source> instead.
  • controls: A boolean value to display controls for this element.
  • autoplay: A boolean element to autoplay when rendered.
  • loop: A boolean element to loop playback.
<source> HTML5 element used to specify one or more resources within <video>, <audio> <picture>. Using more than one source allows the browser to select the most appropriate resource for rendering, since a provided source may not be supported by the user's browser.
  • src: URL reference to the resource.
  • type: MIME type of the resource.
<embed> This deprecated element can be used to display videos and images. It is a self closing tag.
  • src: The URL to the resource you're embedding.
  • type: This is the MIME of the source.
  • width: Width of the resource in pixels (percentages are not allowed).
  • height: Height of the resource in pixels (percentages are not allowed).
